TIMPGUI

TIMPGUI was a project to implement a java-based cross-platform, free and open-source GUI to TIMP. It was to be a full interface to TIMP, and allow for interactive exploration of data and model fit.

It was implemented by Joris Snellenburg , Katharine Mullen, and Sergey Laptenok and renamed to Glotaran.

Learn more about Glotaran on glotaran.org or its github project page: glotaran-legacy.
