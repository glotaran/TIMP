Installing R

    Go to CRAN and install R. You must have R-2.5.0 or higher to use TIMP.

Installing TIMP, stable version from CRAN

    1. start R
    2. execute the command
      install.packages("TIMP")

Installing TIMP, bleeding edge version, from R-Forge

    1. start R
    2. execute the command
      install.packages("TIMP", repos="http://r-forge.r-project.org")
